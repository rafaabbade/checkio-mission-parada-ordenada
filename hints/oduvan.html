Qual pode ser o primeiro passo para resolver esse problema?

Como vamos precisar conferir os elementos de uma lista, recupere um exercício da ilha de iniciação que utilize o loop for, como o "Soma de itens impares de uma lista"


Caso pense em uma solução na qual seja necessário saber a posição de cada um dos elementos, talvez seja melhor utilizar o while. Segue um link para um exercício que utiliza o while: "Contagem par"

Como posso saber a posição de um item na lista?

Será necessário usar o índice/index do item. Por exemplo, para saber o valor do primeiro elemento da lista, usamos nomedalista[0], já o segundo será nomedalista[1].

Não se esqueça que o índice/index sempre começa a contar do zero.

Como eu posso comparar um valor com o próximo ou anterior?

Utilize um contador para tal. Por exemplo: 

                nomedalista[i] < nomedalista[i+1]
